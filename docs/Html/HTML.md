#### 文本标签

+ 加粗、斜体、删除线、下划线

  ```html
  <strong></strong>
  <b></b>
  <em></em>
  <i></i>
  <del></del>
  <s></s>
  <ins></ins>
  <u></u>
  ```

#### 锚点链接

+ `a` 超链接标签

  ```html
  <!-- 形成锚点-->
  <a href='#location'></a> 
  <h1 id='location'></h1>
  ```

#### `Table`

+ 表格属性

  ```html
  <!--th 表头单元格 colpadding td内容和边界距离 colspacing 两个td之间的距离 rowspan 跨行 colspan 跨列-->
  <table>
      <thead><tr><th></th><tr></thead>
      <tbody><tr><td></td></tr></tbody>
  </table>
  ```

#### `lable`

+ 标签 `for` 与表单 `id` 对应

  ```html
  <lable for='sex'></lable>
  <input id='sex' type='radio'/>
  ```
